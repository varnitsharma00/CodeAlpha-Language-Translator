# CodeAlpha Language Translator

This is a simple Language Translator project created using HTML and JavaScript.

## Features
- Enter text for translation
- Select source and target languages
- Uses a free translation API
- Displays translated text
- Copy and text-to-speech features

## Technologies Used
- HTML
- CSS
- JavaScript
- Free Translation API (MyMemory)

## How to Run
1. Download or clone the repository
2. Open `translator.html` in a web browser
3. Enter text and translate
